## How to run the CLI

### Using sbt

In an sbt console:

```bash
kata          # To get the documentation
kata my-ip    # To get your public IP address
````

### Using a native binary

```bash
sbt nativeImage # To build the native binary
./kata          # To get the documentation
./kata my-ip    # To get your public IP address
```
